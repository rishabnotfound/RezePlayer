RezePlayer

A standalone HLS video player built with React

View Demo GitHub

Features

Multiple Sources

Support for HLS and MP4 with seamless switching between sources

Subtitles

Multi-language subtitle support with customizable styling

Watch Party

Synchronized viewing experience with friends

Chromecast

Cast to any Chromecast-enabled device

Quick Start

1. Include CSS and JS

<link rel="stylesheet" href="./dist/assets/style.css">
<script src="./dist/rezeplayer.iife.js"></script>

2. Add Container

<div id="player"></div>

3. Initialize Player

RezePlayer.make('#player', {
  title: 'My Video',
  servers: [
    {
      name: 'Server 1',
      url: 'https://example.com/video.m3u8',
      type: 'hls'
    },
    {
      name: 'Server 2',
      url: 'https://example.com/video.mp4',
      type: 'mp4'
    }
  ],
  subtitles: [
    {
      name: 'English',
      language: 'en',
      src: 'https://example.com/subtitles-en.srt',
      default: true
    }
  ],
  autoPlay: true,
  volume: 1,
  startTime: 0,
  enableWatchParty: true
});

API Reference

Options

servers Array (required) - List of video sources

title String - Video title displayed in player

subtitles Array - List of subtitle tracks

autoPlay Boolean - Auto-play on load (default: true)

volume Number - Initial volume 0-1 (default: 1)

startTime Number - Start position in seconds (default: 0)

enableWatchParty Boolean - Enable watch party feature (default: true)

Methods

const player = RezePlayer.make('#player', options);

// Control playback
player.play();
player.pause();

// Seek to time
player.seek(120); // seek to 2 minutes

// Set volume
player.setVolume(0.5); // 50% volume

// Destroy player
player.destroy();

Full Page Mode

For a full-page player experience, add the data-full attribute to your HTML tag:

<html lang="en" data-full>

Note: The player uses scoped CSS to avoid conflicts with your page styles. All styles are contained within the player container.
